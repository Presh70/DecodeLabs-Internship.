# Sales Analytics Dashboard

## Overview

This project analyzes an e-commerce sales dataset using Microsoft Excel.

The workflow includes:

- Data Cleaning
- Data Validation
- Summary Statistics
- Exploratory Data Analysis (EDA)
- Dashboard Design
