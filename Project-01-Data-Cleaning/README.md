# Project 1: Data Cleaning and Preparation

## Objective

The objective of this project was to clean and prepare an e-commerce dataset for analysis by improving data quality, consistency, and usability.

## Dataset Description

The dataset contains customer orders, product information, pricing details, payment methods, order statuses, and marketing sources.

## Data Cleaning Tasks Performed

* Standardized column headers
* Converted dates to YYYY-MM-DD format
* Checked for duplicate records
* Replaced missing CouponCode values with "Nil"
* Standardized TotalPrice formatting
* Validated dataset consistency

## Deliverables

* Cleaned Dataset
* Data Profile Sheet
* Cleaning Log Sheet
* Project Overview Document

## Tools Used

* Google Sheets
* Microsoft Excel

## Outcome

The dataset was successfully transformed into a clean and analysis-ready format suitable for reporting and business intelligence tasks.
